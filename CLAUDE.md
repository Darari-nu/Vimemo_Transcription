# Vimemo Transcription プロジェクト

## プロジェクト概要
音声メモの文字起こしシステム

## 要件定義

### 基本機能
- [x] VimeoのURLを入力
- [ ] Vimeo動画の字幕データ抽出（JSコード実行）
- [ ] 字幕テキストの整形・表示
- [ ] テキストのコピー・ダウンロード機能

### 技術要件（新アプローチ）
- [ ] Webアプリケーション（フロントエンド中心）
- [ ] Vimeo埋め込み機能
- [ ] JavaScript字幕抽出コード実行
- [ ] テキスト整形・表示機能
- [ ] レスポンシブデザイン

### 字幕抽出ロジック（提供済み）
```javascript
// Vimeoの字幕トラックから直接テキストを抽出

// 音声処理不要、APIコスト不要
```
